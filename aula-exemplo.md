## (CONTEÚDO) O que é um algoritmo

Algoritmo é uma sequência **finita e ordenada** de passos para resolver um problema.

Exemplos do cotidiano:
- Receita de bolo
- Instruções para montar um móvel
- Passo a passo para atravessar a rua

---

## (CONTEÚDO) Estrutura básica de um algoritmo

Todo algoritmo possui:

- **Entrada**: dados recebidos  
- **Processamento**: operações realizadas  
- **Saída**: resultado final  

---

## (EXEMPLOS)

```text
Algoritmo CalcularMedia
  leia nota1
  leia nota2
  media ← (nota1 + nota2) / 2
  escreva media
```

---

## (DÚVIDAS)

**1) Todo algoritmo precisa ter saída?**  
Não necessariamente. Alguns algoritmos apenas executam ações internas, mas na prática quase sempre há algum tipo de resultado observável.

**2) Qual a diferença entre algoritmo e programa?**  
Algoritmo é a ideia/solução.  
Programa é a implementação dessa ideia em uma linguagem.
