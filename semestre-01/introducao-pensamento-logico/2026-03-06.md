## Questão 1 — p: Está frio / q: Está chovendo

| Item | Simbólico      | Linguagem corrente                                                  |
|------|----------------|---------------------------------------------------------------------|
| a    | ¬p             | Não está frio.                                                      |
| b    | p ∧ q          | Está frio e está chovendo.                                          |
| c    | p ∨ q          | Está frio ou está chovendo.                                         |
| d    | q ↔ p          | Está chovendo se, e somente se, estiver frio.                       |
| e    | p → ¬q         | Se está frio, então não está chovendo.                              |
| f    | p ∨ ¬q         | Está frio ou não está chovendo.                                     |
| g    | ¬p ∧ ¬q        | Não está frio e não está chovendo.                                  |
| h    | p ↔ ¬q         | Está frio se, e somente se, não estiver chovendo.                   |
| i    | p ∧ ¬q → p     | Está frio e não está chovendo; então, está frio.                    |

## Questão 2 — p: Jorge é rico / q: Carlos é feliz

| Item | Simbólico      | Linguagem corrente                                                  |
|------|----------------|---------------------------------------------------------------------|
| a    | q → p          | Se Carlos é feliz, então Jorge é rico.                              |
| b    | p ∨ ¬q         | Jorge é rico ou Carlos não é feliz.                                 |
| c    | q ↔ ¬p         | Carlos é feliz se, e somente se, Jorge não for rico.                |
| d    | ¬p → q         | Se Jorge não é rico, então Carlos é feliz.                          |
| e    | ¬¬p            | Jorge é rico.                                                       |
| f    | ¬p ∧ q → q     | Jorge não é rico e Carlos é feliz; então, Carlos é feliz.           |

## Questão 3 — p: Cláudio fala inglês / q: Cláudio fala alemão

| Item | Simbólico      | Linguagem corrente                                                        |
|------|----------------|---------------------------------------------------------------------------|
| a    | p ∨ q          | Cláudio fala inglês ou fala alemão.                                       |
| b    | p ∧ q          | Cláudio fala inglês e fala alemão.                                        |
| c    | p ∧ ¬q         | Cláudio fala inglês e não fala alemão.                                    |
| d    | ¬p ∧ ¬q        | Cláudio não fala inglês e não fala alemão.                                |
| e    | ¬¬p            | Cláudio fala inglês.                                                      |
| f    | ¬(¬p ∧ ¬q)     | Não é o caso que Cláudio não fala inglês e não fala alemão.               |

## Questão 4 — p: João é gaúcho / q: Jaime é paulista

| Item | Simbólico      | Linguagem corrente                                                             |
|------|----------------|--------------------------------------------------------------------------------|
| a    | ¬(p ∧ ¬q)     | Não é o caso que João é gaúcho e Jaime não é paulista.                          |
| b    | ¬¬p            | João é gaúcho.                                                                 |
| c    | ¬(¬p ∨ ¬q)    | Não é o caso que João não é gaúcho ou Jaime não é paulista.                     |
| d    | p → ¬q         | Se João é gaúcho, então Jaime não é paulista.                                  |
| e    | ¬p ↔ ¬q        | João não é gaúcho se, e somente se, Jaime não for paulista.                    |
| f    | ¬(¬q → p)      | Não é o caso que, se Jaime não é paulista, então João é gaúcho.                |

## Questão 5 — p: Marcos é alto / q: Marcos é elegante

| Item | Linguagem corrente                                           | Simbólico          |
|------|--------------------------------------------------------------|--------------------|
| a    | Marcos é alto e elegante.                                    | p ∧ q              |
| b    | Marcos é alto, mas não é elegante.                           | p ∧ ¬q             |
| c    | Não é verdade que Marcos é baixo ou elegante.                | ¬(¬p ∨ q)          |
| d    | Marcos não é nem alto nem elegante.                          | ¬p ∧ ¬q            |
| e    | Marcos é alto ou é baixo e elegante.                         | p ∨ (¬p ∧ q)       |
| f    | É falso que Marcos é baixo ou que não é elegante.            | ¬(¬p ∨ ¬q)         |

## Questão 6 — p: Suely é rica / q: Suely é feliz

| Item | Linguagem corrente                                           | Simbólico          |
|------|--------------------------------------------------------------|--------------------|
| a    | Suely é pobre, mas feliz.                                    | ¬p ∧ q             |
| b    | Suely é rica ou infeliz.                                     | p ∨ ¬q             |
| c    | Suely é pobre e infeliz.                                     | ¬p ∧ ¬q            |
| d    | Suely é pobre ou rica, mas é infeliz.                        | (¬p ∨ p) ∧ ¬q      |

## Questão 7 — p: Carlos fala francês / q: Carlos fala inglês / r: Carlos fala alemão

| Item | Linguagem corrente                                                               | Simbólico              |
|------|----------------------------------------------------------------------------------|------------------------|
| a    | Carlos fala francês ou inglês, mas não fala alemão.                              | (p ∨ q) ∧ ¬r           |
| b    | Carlos fala francês e inglês, ou não fala francês e fala alemão.                 | (p ∧ q) ∨ (¬p ∧ r)     |
| c    | É falso que Carlos fala francês, mas não fala alemão.                            | ¬(p ∧ ¬r)              |
| d    | É falso que Carlos fala inglês ou alemão, mas não fala francês.                  | ¬((q ∨ r) ∧ ¬p)        |
